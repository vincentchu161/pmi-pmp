# Mindmap: Stakeholder Management

- **Stakeholder / Bên liên quan**
  - Individual, group, organization / Cá nhân, nhóm, tổ chức
  - Affected by project / Bị ảnh hưởng bởi dự án
- **Stakeholder Management Steps / Các bước quản lý**
  - Identify / Nhận diện: List, analyze / Liệt kê, phân tích
  - Plan / Lập kế hoạch: Strategy / Chiến lược
  - Manage / Quản lý: Communicate, resolve, build / Giao tiếp, giải quyết, xây dựng
  - Monitor / Giám sát: Track, adapt / Theo dõi, điều chỉnh
- **Analysis Tools / Công cụ phân tích**
  - Power/Interest Grid / Ma trận quyền lực/quan tâm
  - Stakeholder Register / Sổ đăng ký bên liên quan
  - Salience Model / Mô hình salience
  - Stakeholder Cube / Khối lập phương stakeholder
- **Role in Success / Vai trò thành công**
  - Reduce risk / Giảm rủi ro
  - Increase satisfaction / Tăng sự hài lòng
  - Ensure project success / Đảm bảo thành công dự án
- **Exam Tips / Mẹo thi**
  - Identify all early / Nhận diện sớm
  - Use analysis tools / Sử dụng công cụ phân tích
  - Prioritize key stakeholders / Ưu tiên bên liên quan chính
